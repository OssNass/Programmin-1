

## Requirements

- [Node.js](https://nodejs.org/) <= 22.0.0
- yarn

## Mise tool
- Use compatibile node version
```bash
mise use --global node@22
```

- Return to system node version
```bash
mise use --global node@system
```

## Install

```bash
cd themes/slidev-theme-penguin-rtl
yarn install

cd ../..
yarn install
```

## Start

```bash
yarn run dev
```
